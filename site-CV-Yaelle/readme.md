Pour mon site CV, je suis passée par plusieurs étapes :

choisir des couleurs qui me correspondent ;

choisir une structure de CV à modifier sous forme de site, et créer des classes pour chaque item que j’ajoute afin de pouvoir les appeler et les modifier individuellement ;

j’ai choisi d’avancer partie par partie et de créer le responsive à la fin de chaque partie pour être sûre de ne rien casser ;

j’ai ensuite décidé d’un design type à appliquer à chaque partie, en accord avec le dégradé de ma barre de navigation ;

certaines classes possèdent un numéro (1, 2, …) afin de les appeler individuellement et d’adapter les couleurs au responsive en fonction du nombre de colonnes ;

j’ai préféré faire un menu burger sur mobile pour rendre la navigation plus simple ;

je n’ai pas encore de portfolio, donc il n’apparaît pas dans le site ;

pour le JavaScript, je me suis inspirée de travaux déjà réalisés sur internet.

j'ai travaillé avec un document test à coté afin de ne pas casser mon code et rajouter les éléments fonctionnels à mon code final
